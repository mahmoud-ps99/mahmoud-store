## وصف المشروع
موقع تجاري مصغر يتيح للمستخدمين تصفح المنتجات، إضافتها إلى سلة التسوق، وإتمام عملية الشراء. يدعم الموقع إدارة المنتجات والطلبات وبيانات العملاء.

## فكرة المشروع
فكرة المشروع تقوم على توفير منصة بسيطة للتسوق الإلكتروني تناسب المتاجر الصغيرة، مع لوحة تحكم للمدير لإدارة المنتجات والمخزون ومتابعة الطلبات.
* عرض المنتجات مع الصور والأسعار.
* إدارة سلة التسوق وإتمام الطلبات.
* لوحة تحكم للمدير لمتابعة المبيعات.

## طريقة تشغيله

1. استنساخ المشروع:
git clone https://github.com/mahmoud-ps99/mahmoud-store.git

2. إعداد قاعدة البيانات:
افتح `phpMyAdmin` وأنشئ قاعدة بيانات باسم `mahmoud_db` ثم استورد ملف `database.sql`.

3. تشغيل المشروع:
افتح `index.php` مباشرة في المتصفح عبر السيرفر المحلي: `http://localhost/mahmoud_project/index.php`.
